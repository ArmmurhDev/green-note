# **App Name**: GreenNotes

## Core Features:

- Create Note: Allow users to create a new note with a title and body text.
- Edit Note: Enable users to edit the content of existing notes.
- Tagging: Let users add tags to notes for better categorization and organization.
- Search: Implement a search function to find notes by keywords or tags.
- AI-Powered Tag Suggestions: Uses an AI tool to suggest relevant tags based on note content to enhance organization.

## Style Guidelines:

- Background color: Light pastel green (#E8F8EF) for a calm and soothing feel. Heavily desaturated from the primary, this establishes the light color scheme requested.
- Primary color: Soft green (#90EE90) to align with the pastel green and white scheme.
- Accent color: Light yellow-green (#A8E45C), a brighter, more saturated color, that's analogous to the soft green of the primary color, approximately 30 degrees to the 'left' on the color wheel.
- Font: 'PT Sans', a humanist sans-serif font suitable for both headlines and body text.
- Use simple, outlined icons for actions like creating, editing, and tagging notes.
- Clean and minimalistic layout to prioritize content and readability.
